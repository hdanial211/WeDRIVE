## 6. Abstract

### ABSTRACT

The rapid growth of the tourism and transportation sectors in Melaka, Malaysia, has increased the demand for efficient, secure, and modern car rental services. Conventional car rental platforms often suffer from limited vehicle inspections, lack of intuitive customer interaction, and slow data processing. To address these issues, this project introduces **WeDRIVE**, an AI-enabled car rental system designed with a premium web-based interface, a 360-degree virtual vehicle inspection module, and a cloud pangkalan data structure. Developed using HTML, CSS, Javascript, and Supabase (PostgreSQL), WeDRIVE implements advanced frontend features including dynamic calendar blocking (Flatpickr), custom-designed modals, and multi-language support. A key highlight is the integration of an AI support chatbot that provides automated passenger guidance and intelligent route planning in Melaka. The database was progressively migrated from local JSON mockups to a real-time cloud-hosted PostgreSQL system. Authentication is secured using Supabase Auth, integrating both standard credentials and Google OAuth 2.0 with a dedicated user profile completion and verification workflow to ensure regulatory compliance. The administrative panel features sorting, search, and dynamic date range filters for booking history. The final implementation was deployed to Vercel, demonstrating low-latency page loads and fully mobile-responsive layouts. The results indicate that WeDRIVE successfully improves the car booking process, enhances customer trust through virtual inspections, and provides rental operators with a streamlined management dashboard.

**Keywords:** Car Rental System, Artificial Intelligence, Supabase PostgreSQL, 360-Degree Virtual View, Web Application.

## 7. Abstrak

### ABSTRAK

Pertumbuhan pesat dalam sektor pelancongan dan pengangkutan di Melaka, Malaysia, telah meningkatkan permintaan bagi perkhidmatan sewa kereta yang cekap, selamat dan moden. Platform sewa kereta konvensional sering menghadapi masalah pemeriksaan kenderaan yang terhad, kekurangan interaksi pelanggan yang intuitif dan pemprosesan data yang perlahan. Bagi menangani masalah ini, projek ini memperkenalkan **WeDRIVE**, sistem sewa kereta berdaya AI yang direka dengan antara muka web premium, modul pemeriksaan kenderaan maya 360 darjah, dan struktur pangkalan data awan. Dibangunkan menggunakan HTML, CSS, Javascript dan Supabase (PostgreSQL), WeDRIVE melaksanakan ciri-ciri frontend termaju termasuk sekatan kalendar dinamik (Flatpickr), modal rekaan khas, dan sokongan dwi-bahasa. Sorotan utama projek ini adalah integrasi chatbot sokongan AI yang menyediakan panduan pelanggan automatik serta cadangan laluan pelancongan pintar di Melaka. Pangkalan data telah dipindahkan secara progresif daripada simulasi JSON tempatan ke sistem PostgreSQL awan yang dihoskan secara masa nyata. Pengesahan pengguna dilindungi menggunakan Supabase Auth, menyepadukan kelayakan standard dan Google OAuth 2.0 dengan aliran kerja melengkapkan serta mengesahkan profil pengguna bagi mematuhi peraturan keselamatan. Panel pentadbiran dilengkapi dengan fungsi penyusunan data, carian, dan penapis julat tarikh dinamik untuk sejarah tempahan. Implementasi akhir dideploy ke Vercel, menunjukkan pemuatan halaman dengan latensi rendah dan reka letak responsif mudah alih sepenuhnya. Hasil kajian menunjukkan bahawa WeDRIVE berjaya menambah baik proses tempahan kereta, meningkatkan kepercayaan pelanggan melalui pemeriksaan maya, dan memberikan pengendali sewa panel pengurusan yang sistematik.

**Kata Kunci:** Sistem Sewa Kereta, Kecerdasan Buatan, Supabase PostgreSQL, Pandangan Maya 360 Darjah, Aplikasi Web.
